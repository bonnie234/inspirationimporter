Built for visual research, audits, and rebuilds.

Inspiration Importer helps design teams quickly collect useful visual assets from live websites without manually saving images one by one.

Paste a URL

Use a public website URL or a direct image URL. The plugin scans for images, SVGs, icons, logos, and visual assets.

Preview and filter

Review assets in a grid, filter by format, hide tiny assets, hide unavailable previews, and open original source links.

Import to Figma

Select visible assets and import them directly onto your canvas. Quality settings keep large images Figma-safe.

Install the private beta

Download the beta folder, then import the plugin manifest in Figma Desktop.

Download beta ZIP
  1. Download and unzip the beta folder.
  2. Open Figma Desktop.
  3. Go to Plugins → Development → Import plugin from manifest…
  4. Select the manifest.json file from the unzipped folder.
  5. Run Plugins → Development → Inspiration Importer.

Good test URLs

Try these first to confirm the plugin is working.

Known limitation

Some websites block extraction. If that happens, the plugin will say:

This site blocks extraction. Try another public page or use a direct image URL instead.

That means the website refused extraction. The plugin is still working.
