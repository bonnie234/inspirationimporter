Get started in minutes

Download the ZIP, unzip it, then import the manifest in Figma Desktop.

  1. Download and unzip the beta folder.
  2. Open Figma Desktop.
  3. Go to Plugins → Development → Import plugin from manifest…
  4. Select the manifest.json file.
  5. Run Plugins → Development → Inspiration Importer.
Download ZIP

No local server or Terminal needed. The scraper backend is hosted.

Known limitation

Some websites block extraction. When that happens, the plugin will say:

This site blocks extraction.

Try another public page or use a direct image URL instead.
